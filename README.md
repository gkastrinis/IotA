# IotA

System requirements
-------------------

* Java Developer Kit version 8 or newer

Installation & Testing
----------------------

```bash
git clone git://github.com/gkastrinis/iota.git
cd iota
./gradlew run -Pargs=src/test/resources/720.json
```
